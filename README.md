# mra-innovation-co2
Short Summary: Meta Regression Analysis of 49 econometric studies to quantify the impact of innovation investments on CO2 levels.

## Quick start
1. Put cleaned dataset in `data/clean/` named `AGGIUNGEREEE`.
2. Open Stata and run `AGGIUNGERE NOME DO FILE`.
3. Results (tables and graphs) will be in `output/`.

## Requirements
- Stata 16+
- estout/esttab (Stata package)
